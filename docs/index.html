Aprendizagem por Reforço Relacional

Aluno: Thiago Yukio Sikusawa

Orientadora: Leliane Nunes de Barros

Resumo

Dado um problema que queremos resolver com aprendizagem com reforço, diversos dos algoritmos usados necessitam acumular bastante experiência até conseguir uma performance boa. Por causa disso, não é incomum que seja preciso deixar o algoritmo rodando por horas, ou até dias, para conseguir resolver o problema bem. Assim, este trabalho visa explorar uma classe de algoritmos de aprendizagem por reforço, chamados de algoritmos de aprendizagem por reforço relacional, para analisar as suas eficiências em relação à quantidade de experiência e quantidade de tempo preciso até ter uma performace boa, comparado com algoritmos mais tradicionais.
